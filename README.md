#### 快速上手

- 本项目需要安装`Gatsby`与`Node`，如果没有安装的话，可以参考以下步骤进行安装。
- `node`版本需要`=>18.0.0`

#### 📦 Node >= 18
>windows 安装 node

[下载node](https://nodejs.org/download/release/v18.0.0/)

>mac 安装 node
``` bash
brew install node@18
```

#### 📦 Gatsby
``` bash
npm install -g gatsby-cli
```

#### 安装依赖
``` bash
npm i
# or
yarn
```

#### 启动项目
``` bash
npm run dev
# or
yarn dev
```

#### 打包
``` bash
npm run build
# or
yarn build
```
